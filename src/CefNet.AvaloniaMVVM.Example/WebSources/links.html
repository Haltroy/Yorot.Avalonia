Requires Arguments

Use these with arguments. They don't work without arguments.

• yorot://certerror?u=[Address] - Handles the Certificate Error Page.

• yorot://search/?q=[Text] - Searches on the internet with applied Search Engine.

• yorot://error/?e=[Error Code]?u=[Address] - Handles Error Pages.

• yorot://noint/?u=[Address] - Handles the No Internet Page.
